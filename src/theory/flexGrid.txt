# Ultimate CSS Layout Cheat Sheet: Flexbox + Grid

---

## 1️⃣ FLEXBOX – One-Dimensional Layout

Analogy: Flexbox = ants marching in a line (row or column).
- Works along **one axis** at a time.
- Perfect for navigation bars, toolbars, buttons, and small linear layouts.

### Parent Container
.container {
  display: flex;             /* enables flex layout */
  flex-direction: row;       /* row | row-reverse | column | column-reverse */
  justify-content: flex-start; /* align items along main axis */
  align-items: stretch;      /* align items along cross axis */
  flex-wrap: nowrap;         /* wrap | wrap-reverse | nowrap */
}

### Child Item Example
.item {
  flex: 1;                   /* grow/shrink/basis shorthand */
  align-self: flex-start;    /* override cross-axis alignment */
}

Memory Trick: Flex flows, justify moves, align stacks, wrap breaks.

---

## 2️⃣ CSS GRID – Two-Dimensional Layout

Analogy: Grid = chessboard or map with rows and columns.
- Handles rows AND columns simultaneously.
- Perfect for page layouts, galleries, and card systems.

### Parent Container
.grid-container {
  display: grid;
  grid-template-columns: 1fr 2fr 1fr;  /* column widths */
  grid-template-rows: 100px 200px;     /* row heights */
  gap: 10px;                            /* spacing between cells */
}

### 1. Basic Cell Placement
.item1 {
  grid-column: 2 / 4;   /* start at column 2, end before column 4 */
  grid-row: 1 / 3;      /* start at row 1, end before row 3 */
}

### 2. Fractional Units & Repeat
grid-template-columns: repeat(3, 1fr);                /* 3 equal columns */
grid-template-columns: repeat(3, minmax(100px, 1fr)); /* min 100px, flexible */

- fr = fraction of remaining space
- repeat() = shorthand
- minmax(min, max) = responsive columns

### 3. Alignment
justify-items: center;  /* horizontal alignment inside cell */
align-items: center;    /* vertical alignment inside cell */
place-items: center;    /* shorthand for both */

Whole grid alignment:
justify-content: center;
align-content: center;

### 4. Grid Template Areas
.grid-container {
  display: grid;
  grid-template-areas:
    "header header header"
    "sidebar main main"
    "footer footer footer";
  grid-template-columns: 1fr 2fr 2fr;
  grid-template-rows: 50px 200px 50px;
}

.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main   { grid-area: main; }
.footer { grid-area: footer; }

### 5. Auto Placement
grid-auto-flow: row | column | dense;

### 6. Memory Tips
- Think “chessboard”: rows + columns
- fr = fraction of space
- gap = spacing between cells
- grid-area = semantic labels
- span = extend across multiple cells

---

## 3️⃣ Flexbox vs Grid Quick Comparison

Feature | Flexbox | Grid
------- | ------- | ----
Dimensions | 1D (row OR column) | 2D (row AND column)
Best For | Toolbar, nav bar, linear items | Page layout, gallery, complex layouts
Item Control | flex, align-self | grid-column, grid-row, justify/align
Wrapping / Flow | flex-wrap | grid-auto-flow
Spacing | justify-content, align-items | gap, justify-items, align-items
Fractions | flex-grow, flex-shrink | fr units, minmax

## 4️⃣ Example: Chessboard Using CSS Grid

HTML:
<div class="chessboard">
  <!-- 64 divs -->
  <div></div><div></div> ... repeat until 64
</div>

CSS:
.chessboard {
  display: grid;
  grid-template-columns: repeat(8, 50px); /* 8 columns */
  grid-template-rows: repeat(8, 50px);    /* 8 rows */
  gap: 0;                                 /* no spacing for exact squares */
}

.chessboard div {
  width: 50px;
  height: 50px;
}

/* Alternating colors */
.chessboard div:nth-child(8n+1),
.chessboard div:nth-child(8n+3),
.chessboard div:nth-child(8n+5),
.chessboard div:nth-child(8n+7) {
  background-color: black;
}
.chessboard div:nth-child(8n+2),
.chessboard div:nth-child(8n+4),
.chessboard div:nth-child(8n+6),
.chessboard div:nth-child(8n+8) {
  background-color: white;
}

/* Responsive version using fractions */
.chessboard {
  grid-template-columns: repeat(8, 1fr);
  grid-template-rows: repeat(8, 1fr);
  width: 400px;
  height: 400px;
}

Analogy: Each square = a chessboard cell; CSS Grid handles 8x8 layout, colors alternate mathematically.

---

## 5️⃣ General Tips & Analogies

- Flexbox: “Line of ants marching” → use for 1D layout.
- Grid: “Chessboard or map” → use for 2D layout.
- Memory:
  - justify- → main axis / horizontal in cell
  - align- → cross axis / vertical in cell
  - fr = flexible fraction
  - gap = spacing
  - span = extend multiple cells
- Combined: Grid for layout, Flexbox for items inside cells.
